Linux常用命令

1、列出文件————ls

  • -a 列出全部文件
  • -t 按时间列出文件
  • -l列出文件全部信息
  • -S按文件大小信息

2、进入文件————cd

  • /进入主目录
  • ~进入根目录
  • -进入上次路径
  • ..进入上级目录
  • /xxx/xxx 绝对路径进入
  • ./xxx相对路径进入

3、查看当前路径——-pwd

  • -p查看软连接的实际路径

4、创建文件夹———mkdir

  • -p /xxx/xxx 若路径中的某些目录尚不存在,加上此选项后，系统将自动建立好那些尚不在的目录，即一次可以建立多个目录

5、删除文件夹———rmdir

  • -p xxx/xxx 删除文件夹后如果为空则全删除

6、删除文件————rm

  • -rf xxx 删除目录和所有文件并且不询问
  • -i xxx删除前询问

7、重命名和移动——-mv

  • mv file1 file2 把file1命名为file2
  • mv file1 file2 file3 /dir 把文件123移动到dir中
  • mv * ../ 把当前所有文件上移一级

8、复制文件————cp

  • cp file dir 复制文件到目录
  • -r 复制目录及其文件到目标目录
  • -s x.txt link_x.txt 创建快捷方式

9、显示创建合并文件–cat

  • cat file 显示文件内容
  • cat > filename 创建一个文件
  • cat dir1 dir2 > dir3合并dir12为dir3

10、查找

whereis

  • 该指令只能用于查找二进制文件、源代码文件和man手册页

locate

  • 用于一般文件的定位
  • -l num 显示多少行信息
  • xx/xx 查找xx目录下信息

find

  • 遍历硬盘查找文件
  • /dir xxx 指定目录下查找文件

grep

  • 筛选，一般配合管道操作

11、权限修改———–chmod

  • chmod r file增加读权限
  • chmod w file增加写权限
  • chmod x file增加
  • chmod -x file

12、显示磁盘空间——df

  • -a全部文件
  • -h便于阅读的形式比如KB、MB

13、显示文件大小——du

  • -a 全部文件
  • -h以易于读的方式显示

14、查看进程———–ps

  • a显示所有进程
  • -f 详细信息

15、中止进程———–kill

16、创建文件———–touch

17、下载

yum

  • -y install 不需要确认下载，centos下的下载指令

apt

  • -y install 不需要确认下载，ubuntu下的下载指令

18、书写文件———–vim

书写模式

  • i

指令模式

  • : 进入指令模式

  • 保存———–wq

    退出不保存—-q!

退出模式

  • Esc

19、其他指令

查看历史命令

  • history

注释命令（方便回查）

  • #

换行继续输入指令

  • \+entry

并行输入指令

  • &&
  • ;

查看版本号

  • cat /etc/centos-release

访问主机xxx端口

  • curl localhost:xxx

下所有文件

  • dir/* dir

设置初始root密码

  • sudo passwd

进入root账户

  • su

时间戳

  • date +%s

定时任务(用于自动化执行脚本)
